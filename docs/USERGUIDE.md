## 1. Getting Started

### 1.1 Creating Your Account

1. **Navigate to the application URL**
2. **Click "Create Account"** on the login screen
3. **Enter your details:**
   - Email address (will be used for login)
   - Password (minimum 6 characters)
4. **Click "Sign Up"**
5. **Verify your email** if email confirmation is enabled

### 1.2 First-Time Setup: Creating Your Organization

After logging in for the first time, you'll see the **Workspace Gate**:

1. **Enter your organization name** (e.g., "Manchester City FC" or "Premier Sports Agency")
2. **Click "Create Organization"**
3. You are now the **Owner** of this organization

### 1.3 Creating Your First Club

With your organization created:

1. **Click "Add Club"** in the Workspace Gate
2. **Fill in club details:**
   - **Name:** Full club name (e.g., "Manchester City FC")
   - **Nickname:** Short name for headlines (e.g., "City")
   - **Slug:** URL-friendly identifier (e.g., "man-city")
   - **Tone Context:** Describe the club's voice (e.g., "Professional, ambitious, winning mentality")
   - **Primary Color:** Main brand color (hex code)
   - **Secondary Color:** Accent color (hex code)
3. **Click "Create Club"**
4. **Select your club** to enter the main application

### 1.4 Understanding the Interface

The application has these main areas:

- **Sidebar (Left):** Navigation, workspace info, user menu
- **Main Content (Center):** Active tab content
- **AI Panel (Right):** Contextual AI assistant

**Navigation Tabs:**
- Dashboard â€” Command center overview
- Squad â€” Player management
- Fixtures â€” Match schedule and reporting
- Content â€” Content pipeline
- Commercial â€” Sponsor management
- Inbox â€” Email management
- Admin â€” Administrative tasks
- Education â€” Learning modules
- Settings â€” Configuration

3. **Modify fields as needed**
4. **Click "Save Changes"**

### 3.5 AI Player Analysis

When you select a player:

2. **AI automatically generates analysis** if not already present
3. **View comprehensive scouting report:**

### 3.6 Generating Player Videos

### 3.7 Tactics Board

## 4. Fixtures & Match Reporting

### 4.1 Viewing Fixtures

1. **Click "Fixtures"** in the sidebar
2. **Toggle views:**
   - **Upcoming:** Scheduled matches
   - **Archive:** Completed matches

### 4.2 Adding a New Fixture

1. **Click "Schedule Fixture"** button in the header
2. **Fill in fixture details:**
   - **Opponent:** Name of opposing team
   - **Date & Time:** Kickoff date and time
   - **Venue:** Home or Away
   - **Competition:** League, cup, or friendly (optional)
3. **Click "Schedule Fixture"**

The new fixture appears in the "Upcoming" tab.

### 4.3 Deleting a Fixture

1. **Hover over a fixture card** in the Upcoming tab
2. **Click "Delete Fixture"** at the bottom of the action panel
3. **Confirm deletion** in the dialog

> **Note:** Deleting a fixture also removes any associated generated content.

### 4.4 Fixture Card Information

Each fixture shows:
- Date and time
- Opponent name
- Venue (Home/Away)
- Competition
- Status (Scheduled/Completed)

### 4.3 Logging a Match Result

This is the **core workflow** for content generation:

**Step 1: Start the Report**
1. Find the match in Fixtures (upcoming or recent)
2. Click **"Generate Report"** button on the fixture card

**Step 2: Enter Match Data**
Fill in the **Match Report Modal**:

- **Score:** Home and Away goals
- **Scorers:** Click "Add Scorer" for each goal
  - Select player
  - Enter minute
  - Add for each goal scored
- **Man of the Match:** Select standout player
- **Match Stats:**
  - Possession %
  - Shots on Target
  - Corners
  - Fouls
- **Key Events:** Describe notable moments (injuries, red cards, etc.)
- **Match Vibe:** Select tone (Dominant Win, Gritty Battle, etc.)
- **Manager Quote:** Enter post-match quote

**Step 3: Generate Content**
1. Click **"Generate Match Report"**
2. AI generates:
   - Full match report article
   - Social media posts (Twitter, Instagram)
   - Fan communication
3. **Review generated content** in preview
4. **Edit if needed**
5. Click **"Confirm & Save"**

**What Gets Created:**
- Fixture updated with result + stats
- Match report content item
- Social post content items
- All items in Content Pipeline for review

### 4.4 Viewing Past Match Reports

1. Go to **Archive** tab
2. Click completed fixture
3. View stored report and content

## 8. Email Inbox

### 8.1 Understanding Inbox Types

PitchSide AI supports two inbox types:

**Master Inbox (Shared):**
- Accessible by all team members
- For club-wide communications
- Connected by Admin or Owner
- Emails visible to entire team

**My Inbox (Personal):**
- Only visible to you
- Your personal professional email
- Only you can connect/view

### 8.2 Connecting an Email Account

**Connect Master Inbox (Admin/Owner only):**
1. Go to **Inbox** tab
2. Ensure **"Master Inbox"** tab is selected
3. Click **"Connect Email"**
4. Choose **Gmail** or **Outlook**
5. Complete OAuth authorization
6. Grant required permissions
7. Connection saved as shared master

**Connect Personal Inbox:**
1. Go to **Inbox** tab
2. Switch to **"My Inbox"** tab
3. Click **"Connect Email"**
4. Choose provider
5. Complete OAuth flow
6. Connection is private to you

### 8.3 Syncing Emails

1. Click **"Sync Now"** button
2. System fetches new emails from provider
3. Emails appear in list
4. Repeat as needed for fresh content

### 8.4 Reading & Replying to Emails

**Viewing an Email:**
1. Click email in list
2. See full message body
3. AI analyzes sentiment (shown as badge)

**AI Smart Replies:**
1. System generates 3 suggested replies
2. Click a suggestion to use as draft
3. Edit as needed

**Sending a Reply:**
1. Write/edit reply in text box
2. Click **"Send Reply"**
3. Email sent through connected account
4. Reply logged in system

### 8.5 Inbox Precedence

When viewing Master Inbox:
- System checks for **Club Master** connection first
- Falls back to **Org Master** if no club master
- Ensures team always has access to shared communications

## 11. Multi-Tenancy for Agencies

### 11.1 Agency vs Single Club

**Single Club Admin:**
- One organization
- One club
- Direct access to all features

**Agency (Multiple Clubs):**
- One organization
- Multiple clubs
- Switch between clubs
- Centralized management

### 11.2 Managing Multiple Clubs

**Adding a Club:**
1. Click workspace switcher
2. Click "Add Club"
3. Fill club details
4. New club appears in switcher

**Switching Clubs:**
1. Click workspace switcher in sidebar
2. Select desired club
3. App context changes immediately

**Shared vs Club-Specific:**
- **Shared:** Org-level AI settings, org members
- **Club-Specific:** Players, fixtures, content, sponsors

### 11.3 Team Member Roles

**Owner:**
- Full access to everything
- Can manage org settings
- Can add/remove members
- Can delete org

**Admin:**
- Full access to club features
- Can connect master inbox
- Can manage club settings
- Cannot delete org

**Editor:**
- Can create/edit content
- Can log match results
- Can manage players
- Cannot change settings

**Viewer:**
- Read-only access
- Can view all data
- Can use AI assistant
- Cannot modify anything

### 11.4 Inviting Team Members

1. Go to Settings
2. Navigate to "Team" section
3. Click "Invite Member"
4. Enter email address
5. Select role
6. Send invitation

## 12. Troubleshooting

### 12.1 Common Issues

**"Demo mode" message:**
- Supabase not configured
- Contact administrator to set up database

**Can't log in:**
- Check email/password
- Try password reset
- Check if account exists

**Content not saving:**
- Check internet connection
- Verify you have Editor+ role
- Try refreshing the page

**AI generation failed:**
- API key may be invalid
- Check BYOK settings
- Try again after a moment

**Email sync not working:**
- OAuth token may have expired
- Disconnect and reconnect account
- Verify provider permissions

### 12.2 Error Messages

| Error | Meaning | Solution |
|-------|---------|----------|
| "Permission denied" | Insufficient role | Contact admin for role upgrade |
| "Network error" | Connection issue | Check internet, retry |
| "Session expired" | Login timed out | Log in again |
| "Invalid API key" | BYOK key wrong | Update key in settings |
| "Failed to sync" | Email provider issue | Reconnect account |

### 12.3 Getting Help

1. **Use AI Assistant:** Ask questions about features
2. **Check Documentation:** Review this guide
3. **Contact Admin:** For access/permission issues
4. **Report Bug:** Note exact steps to reproduce
